# 💼 Practical No.5 – Web Designing Lab

## 🌐 Resume Webpage using HTML5

This project is part of the **Web Designing Lab Practical** where a **creative and professional resume webpage** is designed using **HTML5 and CSS3**.

The webpage showcases a **personal resume layout** including a **profile image, career objective, education table, skills, projects, hobbies, and hyperlinks**.
The design focuses on **modern UI, attractive colors, and clean structure** to make the webpage visually appealing.

---

## 🎯 Objective

To design a **resume webpage using HTML5** by integrating:

* 🖼 **Image**
* 📋 **Lists**
* 📊 **Table**
* 🔗 **Hyperlinks**

while applying **CSS styling for a modern and attractive layout**.

---

## ✨ Features

* 🧑‍💻 Professional **resume webpage design**
* 🌈 **Animated gradient background**
* 🖼 Profile **image integration**
* 📊 **Education details displayed using table**
* 📋 **Lists for skills, projects, and hobbies**
* 📈 **Skill progress bars**
* 🔗 **Hyperlinks to GitHub / LinkedIn**
* 🎨 Modern **CSS styling and hover effects**
* 📱 Clean and structured **HTML5 layout**
